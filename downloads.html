Windows blockiert die Installation aus „unbekannter Quelle"? Das ist normal – der Installer ist (noch) nicht signiert, aber unbedenklich, wenn du ihn von hier bzw. dem verlinkten GitHub-Release lädst. So geht's: Im blauen Fenster „Der Computer wurde durch Windows geschützt" auf „Weitere Informationen" → „Trotzdem ausführen" klicken. Bleibt die Datei blockiert: Rechtsklick auf die .exe → „Eigenschaften" → bei „Sicherheit" „Zulassen" (Entsperren) anhaken → „OK".
